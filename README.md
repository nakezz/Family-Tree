# Family Tree

## Description

This is a family tree project that uses Prolog to represent and query family relationships.
Uses HTML, CSS, and JavaScript to display the family tree. Also you can you write queries to get information about the family members in the browser.

## How to run

1. Install SWI-Prolog
2. Run `swipl server.pl`
3. Open `http://localhost:8080` in your browser
